Tutorial 0: Creating your development environment

Welcome to the first of my modding tutorials for Minecraft 1.12.2. Currently, whilst Minecraft 1.13 and 1.14 are both released, Forge lags behind and so the modding community is still using 1.12.

In order to create Minecraft mods efficiently, it is important to set up your development environment correctly. Below, I will go through all required downloads, installs as well as how to set up your file structure correctly.

Part 1: Requirements

Java Development Kit

You're studying <insert computing degree here>, figure it out. Although, I will note that you should be using Java 8. Older versions are okay (unless it is more ancient than my nan) but newer versions are not.

IntelliJ IDEA

Almost Minecraft mods are written using IntelliJ IDEA or Eclipse as the build tool that we use, Gradle, is able to automatically setup development environments for those IDEs. Get it for free as a student at jetbrains.com/student

Minecraft Forge

Minecraft Forge is *the* Minecraft modding API. Other APIs do not exist. If you see something claiming to be a Minecraft modding API, ignore it. It doesn't exist.

To create our mod, we need to grab the latest version of the Minecraft Forge src. Head over to the Minecraft Forge downloads page and grab the latest version of the Forge MDK (Mod Development Kit). At the time of writing this is 14.23.5.2847.

But where are we going to extract that zip file? Fear not! NEXT SECTION!

Installing Forge

Putting things in the wrong place is just going to make your life difficult. It's like doing column addition and putting all the digits on a diagonal slant and then wondering why 20+25=270.

If you haven't already, create a folder somewhere on your hard drive (or SSD, you saucy nugget) called Development. Ideally this is in your root folder for easy access via terminal.

Inside that folder, create another folder which is named after your mod (this is the point where you could create a GitHub repository).

Inside of the mod folder, extract the following from the Forge MDK zip:

  • /gradle
  • .gitignore
  • build.gradle
  • gradle.properties
  • gradlew
  • gradlew.bat

To install Forge, open a terminal of your choosing and type the command gradlew setupDecompWorkspace idea

This will download all the stuff you need and create a handy little .ipr file that you can open in IntelliJ to open everything you need.
